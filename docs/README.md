# GovCMS 文档中心

## 文档角色

本目录是 GovCMS 项目的唯一事实来源，用于统一项目定位、分层口径、当前状态、目标范围、实施路线和交付标准。

当前统一口径如下：

- 当前仓库代码已从 `L0 当前基座` 推进到 `M2 CMS 平台共享层` 与 `M3 后台标准版` 的首批实现阶段。
- 首期正式建设目标仍然是 `L1 CMS 平台共享层`、`L2 后台标准版`、`L3 门户标准版`。
- 角色体系、导航、专题、站点管理员已进入产品建模阶段。
- 信创适配和国密能力不是后置专题，而是正式交付基线。
