# Task Plan: GovCMS

## 计划原则

当前项目按以下顺序实施，不允许跳过基础层直接做门户页面或正式交付包装：

1. 统一文档口径
2. 建立 CMS 平台共享层
3. 实现后台标准版
4. 实现门户标准版
5. 完成信创、国密与正式部署交付
